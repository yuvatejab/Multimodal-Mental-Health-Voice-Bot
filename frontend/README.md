# Frontend - Mental Health Voice Bot

React frontend for the Mental Health Voice Bot application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Update `.env` if needed (default works for local development):
```
VITE_API_URL=http://localhost:8000
```

## Running

```bash
npm run dev
```

Visit http://localhost:5173

## Building for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

## Preview Production Build

```bash
npm run preview
```
